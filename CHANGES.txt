0.0.22 (2014-11-20)

* install script added check for pip version for `--pre` flag. added ability to specify wether or not to get pre-released streamer
* added ability to override epoch timestamp via optional `epoch` parameter on `Streamer.log(signal, value[, epoch])`
* added ability to override buffer size before shipping logs via optional constructor parameter `Streamer([buffer_size])`
* added CHANGES.txt to repository to start documenting changes
* added more status code checks to bucket creation to return more descriptive message for incorrect `client_key`
